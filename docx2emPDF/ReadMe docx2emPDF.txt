Instructions for converting a Word file to PDF on a Mac with Clickable Hyperlinks using docx2emPDF

1. Installation of LibreOffice Suite:
Visit LibreOffice's official website (https://www.libreoffice.org) and download the software. Proceed to install it in the default directory as guided by the installation wizard.

2. Converting Word File:
Open the LibreOffice application and simply drag and drop your Word file into the app. If prompted, grant permission for the application to access the file path.

Outcome:
The newly created PDF, complete with live hyperlinks, will be saved in the same folder as the original Word document.
